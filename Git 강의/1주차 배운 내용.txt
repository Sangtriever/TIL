2. 1주차 개념 키워드 

버전관리 : 내용 수정, commit을 통해 버전관리

commit : 파일수정 - 스테이지에 올리기 - 커밋 
	파일을 수정하고, 상세내용을 적기

history : 지금까지 commit 한 내용 (그래프, 설명, 날짜, 작성자, 커밋이 있다)

clone : 원격reop에 있는 내용을 로컬repo로 가져오는것 

tracking : 로컬 repo를 원격repo에 연결하는것

push : 로컬repo -> 원격repo
pull : 원격repo -> 로컬repo
